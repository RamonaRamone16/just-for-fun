Использованные технологии

HTML & CSS & BOOTSTRAP

Сайт верстается с помощью HTML и CSS с элементами Bootstrap. Подключается шрифт, иконки, bootstrap, библиотека jquery.

.NET MVC

Чтобы сделать сайт динамичным необходимо знание технологии .NET MVC Core. Нужно понимать сетевое программирование и как работают http-запросы и -ответы. Также нужно уметь подключать базу данных, это можно сделать с помощью Entity Framework.

C#

И конечно, чтобы все это сделать нужно знать язык программирования C# и понимать принципы ООП.

Как создать статичный сайт?

Шаблон

Нужно представить в голове шаблон, как будет выглядеть сайт. Выбрать дизайн, цвета, шрифт, иконки, а самое главное - концепцию

1

Верстка

Сверстать сайт можно очень быстро. Для этого достаточно изучить пару уроков на ютубе по HTML и CSS, благо их целая куча, и попрактиковаться. Можно подключить к сайту еще библиотеку jquery и bootstap, какие-нибудь шрифты и иконки, тогда сайт будет выглядеть очень современно и красиво.

2

Разместить сайт в интернете

Для этого можно воспользоваться разными беслатными хостами или как в моем случае разместить сайт на GitHub Pages. Второй вариант очень быстрый и удобный. Для этого нужно просто зарегистрироваться в GitHub, создать репозиторий и залить туда сайт.
